Les 3 Niveaux du Web

Surface Web

  • Sites accessibles à tous
  • Indexés par Google
  • 10% du web total
  • Ex: Facebook, Wikipédia

Deep Web

  • Non indexé par Google
  • Bases de données privées
  • 90% du web total
  • Ex: emails, données bancaires

Dark Web

  • Accès via Tor/I2P
  • Anonymat garanti
  • Partie du Deep Web
  • Adresses .onion
